## WASocket

The main socket interface returned by `makeWASocket()`. It provides all methods for interacting with WhatsApp.

```typescript
export type WASocket = ReturnType<typeof makeWASocket>
```

## Core properties

<ResponseField name="ev" type="BaileysEventEmitter" required>
  Event emitter for all WhatsApp events
</ResponseField>

<ResponseField name="ws" type="WebSocket" required>
  Underlying WebSocket connection
</ResponseField>

<ResponseField name="authState" type="AuthenticationState" required>
  Current authentication state and credentials
</ResponseField>

<ResponseField name="user" type="Contact">
  Your own user information
</ResponseField>

<ResponseField name="signalRepository" type="SignalRepositoryWithLIDStore" required>
  Signal protocol repository for encryption
</ResponseField>

## Messaging methods

### sendMessage

Send a message to a chat.

```typescript
sendMessage(
  jid: string,
  content: AnyMessageContent,
  options?: MiscMessageGenerationOptions
): Promise<WAMessage>
```

<ResponseField name="jid" type="string" required>
  JID of the recipient (user or group)
</ResponseField>

<ResponseField name="content" type="AnyMessageContent" required>
  Message content (text, media, etc.)
</ResponseField>

<ResponseField name="options" type="MiscMessageGenerationOptions">
  Additional options (quoted message, ephemeral, etc.)
</ResponseField>

**Example:**

```typescript
// Send text message
await sock.sendMessage('1234567890@s.whatsapp.net', { 
  text: 'Hello!' 
})

// Send image with caption
await sock.sendMessage('1234567890@s.whatsapp.net', {
  image: { url: './photo.jpg' },
  caption: 'Check this out!'
})

// Reply to a message
await sock.sendMessage('1234567890@s.whatsapp.net', 
  { text: 'Thanks!' },
  { quoted: messageObj }
)
```

## Group management methods

### groupCreate

```typescript

  subject: string,

```

<ResponseField name="subject" type="string" required>
  Group name
</ResponseField>

<ResponseField name="participants" type="string[]" required>

**Example:**


### groupParticipantsUpdate

Add, remove, promote, or demote participants.

```typescript

  jid: string,
  participants: string[],
  action: ParticipantAction
): Promise<Array<{ status: string; jid: string }>>
```

<ResponseField name="action" type="ParticipantAction" required>

**Example:**

```typescript
  '123456@g.us',

  'promote'
)
```

## Presence methods

### sendPresenceUpdate

Update your presence.

```typescript
sendPresenceUpdate(
  type: WAPresence,
  toJid?: string
): Promise<void>
```

<ResponseField name="type" type="WAPresence" required>
  `'available'`, `'unavailable'`, `'composing'`, `'recording'`, `'paused'`
</ResponseField>

<ResponseField name="toJid" type="string">
  Specific chat to update presence in
</ResponseField>

**Example:**

```typescript
// Set online
await sock.sendPresenceUpdate('available')

// Show typing in a chat
await sock.sendPresenceUpdate('composing', '1234567890@s.whatsapp.net')

// Stop typing
await sock.sendPresenceUpdate('paused', '1234567890@s.whatsapp.net')
```

## App state sync methods

### chatModify

Modify chat state (archive, pin, mute, etc.).

```typescript
chatModify(
  modification: ChatModification,
  jid: string
): Promise<void>
```

**Example:**

```typescript
// Archive chat
await sock.chatModify({ archive: true }, '1234567890@s.whatsapp.net')

// Pin chat
await sock.chatModify({ pin: true }, '1234567890@s.whatsapp.net')

// Mute chat for 8 hours
await sock.chatModify(
  { mute: Date.now() + 8 * 60 * 60 * 1000 },
  '1234567890@s.whatsapp.net'
)
```
